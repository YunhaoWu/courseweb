This product is a course system. Aiming to implement a online course website like Udemy or Coursera. We have some courses for you to choose and everyone have a price. By adding them to cart, you can purchase and enroll.

We use HTML, CSS, JavaScript for front-end and use PHP for back-end, use Mysql for database.

For run, needing to have PHP installed and a webserver. index.html is the entrance document.
